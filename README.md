# Subscription Customer Analytics Dashboard

## Project Overview

This project analyzes customer subscription data using Excel, SQL, and Power BI. The objective is to uncover customer behavior, identify churn patterns, analyze revenue trends, and provide actionable business insights.

## Business Insights

- Month-to-month contract customers have the highest churn.
- Long-term contracts improve customer retention.
- Electronic check users show higher churn than automatic payment users.
- Fiber optic customers generate high revenue but also have higher churn.
- Long-tenure customers contribute the most revenue.
- Revenue loss is mainly driven by churned customers on short-term contracts.
